To create a Cloud SQL MySQL instance in Google Cloud Platform and perform database operations including:

Step 1 — Open Google Cloud Skills Boost
Go to: https://www.cloudskillsboost.google

 Step 1: Start Qwiklabs Environment
Login to:
https://www.cloudskillsboost.google
Start the lab:
Cloud SQL for MySQL: Qwik Start
Click:
Open Google Cloud Console
Login using temporary lab credentials


🔹 Step 2: Navigate to Cloud SQL
Click Navigation Menu (☰)
Select:
SQL

🔹 Step 3: Create Cloud SQL Instance
Click:
Create Sandbox Instance
Keep default configuration:
Edition → Enterprise
Preset → Sandbox
Database Version → MySQL 8.4


Enter:
Instance Name: mysql-instance
Password: (Any strong password)
Click:
Create Instance

⏳ Wait 3–5 minutes
✅ Checkpoint 2
Instance status should become:
Running


🔹 Step 4: Create Database
Click:
Databases
Enter:
Database Name: studentdb
Keep default settings
Click:
Create

✅ Checkpoint 3
Database studentdb should appear in list.

🔹 Step 5: Connect Using Cloud Shell
Click Cloud Shell icon (Top right)

Wait for terminal to open

Run Command:
gcloud sql connect vipin --user=root
Enter instance password when prompted

✅ Checkpoint 4
Student should see:
mysql>


🔹 Step 6: Select Database
USE studentdb;

🔹 Step 7: Create Table
CREATE TABLE students (
    id INT PRIMARY KEY,
    name VARCHAR(50),
    course VARCHAR(50)
);


🔹 Step 8: Insert Records
INSERT INTO students VALUES (1,'Rahul','Cloud Computing');
INSERT INTO students VALUES (2,'Sneha','Data Science');
INSERT INTO students VALUES (3,'Amit','AI & ML');


🔹 Step 9: Retrieve Records
SELECT * FROM students;
